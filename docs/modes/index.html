Modes

Duels#

Duels modes include: Bridge and Skywars duels. The roles will count you're overall wins in that gamemode, following Hypixel duel's leveling.

Bedwars#

The role system for bedwars is by prestiges(Iron, Gold, Diamond, etc.). The unit of measurement is by using stars, instead of wins. Stars are a much better way of measuring capability and it follows the same system as Hypixel.

Skywars#

Skywars uses the level of the player for the roles. This mode is the most role consuming mode, as it can create infinite roles. However, you can use the -set skywars interval {interval} command to set the intervals of the roles. An interval of 5 is recommended, because it is efficient and clean.
